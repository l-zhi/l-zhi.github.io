浏览器渲染机制

css 都放在头部和body最后，并且后面的 css 延迟 6s, 头部添加一个内链的js

结论：当js放head中的时候，ie,chrome没有差别都是等待css加载完，然后构建render tree。

<!DOCTYPE HTML>
<html>
<head>
<meta http-equiv="content-type" content="text/html; charset=utf-8">
<title> 浏览器渲染机制 </title>
<link rel="stylesheet" href="css/style.css"/>
<script src="js/empty.js"></script>
</head>
<body>
<p class="p1"></p>
<p class="p2"></p>
<p class="p3"></p>
<link rel="stylesheet" href="proxy/style.php"/>
</body>
</html>
